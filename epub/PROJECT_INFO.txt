🎯 项目概述
---------
这是一个完整的 EPUB 电子书处理工具集,参考 PDF skill 设计,提供了
理论指南和实际可用的 Python 脚本,支持 EPUB 文件的读取、创建、编辑、
验证、合并和分割等操作。

📦 项目组成
---------
文档文件 (4个):
  - SKILL.md         主要操作指南(20KB, 400+ 行)
  - README.md        项目说明文档
  - SUMMARY.md       项目创建总结
  - QUICKREF.md      快速参考卡片

核心脚本 (8个):
  - extract_metadata.py   提取元数据(标题、作者、ISBN等)
  - extract_text.py       提取纯文本内容
  - extract_images.py     提取所有图片
  - create_epub.py        从Markdown创建EPUB
  - update_metadata.py    修改元数据
  - validate_epub.py      验证结构完整性
  - merge_epubs.py        合并多个EPUB
  - split_epub.py         按章分割EPUB

辅助工具 (4个):
  - check_env.py      环境检查脚本
  - install.sh        一键安装依赖
  - examples.sh       使用示例集合
  - scripts/README.md 脚本详细说明

🔧 技术栈
---------
Python 库:
  - ebooklib        EPUB核心操作库
  - BeautifulSoup4  HTML解析
  - lxml            XML/HTML解析器

外部工具(可选):
  - Pandoc          格式转换
  - Calibre         ebook-convert

---------
1. 安装依赖:
   cd scripts/ && bash install.sh

2. 检查环境:
   python3 check_env.py

3. 使用脚本:
   python3 extract_metadata.py book.epub
   python3 validate_epub.py book.epub
